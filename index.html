KCC MSIS

MBK의 고객 정비예약 관련 디스플레이를 위한 시스템 구축

  • 기간: 2015.08 ~ 2016.03
  • 발주: MBK(메르세데스 벤츠 코리아)

  • 주요기술:
  • Spring F/W(4.x), XML

  • 주요역할:
  • 단독 개발
  • 고객요구사항부터 분석/설계, 구현까지

[ 혼자서 했던 프로젝트 ]
두 가지 일을 헀다. 처음엔, IOT 연결을 하게 됐다.
다름 아닌 자동차 정비센터 앞에 설치한 카메라에서 보내준 사진과 차량번호를 통해
해당 차량번호가 CRM의 정비예약으로 등록되어 있는지 확인하고,
예약이 등록된 고객이면 예약고객이 왔다고 리셉션 PC에 알림을,
예약이 아니라면, 예약은 아니지만 고객이라든지 입차 알림기능을 구현하였다.
카메라 업체 분이 굉장히 잘 해주셔서 쉽게 연결했던 기억이 있다.

두번째는, MBK의 모 데이터로 저장하고 있는 실제 정비예약 데이터를 연동하여,
리셉션 PC에서 실시간으로 예약현황을 보여주는 것이었다.
내용만은 어렵지 않지만, 독일 회사인 MBK의 모 데이터를 받는 것부터
VPN, NAT 연결을 통해야지만 돼서, 각종 네트워크 이슈가 있어 나를 힘들게 했다.
정비예약 데이터, 즉 개인정보가 들은 데이터여서 처리에 대한 처리도 고민이었다.
결론은, 메모리에만 올려놓자였다. 즉, 캐싱이다.
예약데이터가 그렇게 많지 않을 것이기 때문에 가능했다.
캐싱에 대해서도 이 때 많이 알게되었다.

혼자 고객요구사항부터 분석/설계, 구현까지 진행하고
고객관리 차원에서 진주 등 직접 센터 내 PC에 설정까지 해주고 다녔다.
혼자 다 하고, 고객응대까지.. 꽤 뿌듯했던 경험이었다.

  • 발주: KCC오토그룹

  • 주요기술:

  • 주요역할:
  • 기구축된 솔루션으로 유지보수 및 신규 기능 개발
  • 영엽사원 실시간 실적 모니터링 개발
  • CRM4U 모바일 버전 공통기능 개발

[ 자신감을 많이 얻었던 프로젝트 ]
회사 내 연구소 소속으로 계열사가 기본적으로 사용하던
솔루션을 유지보수 하고 신규기능을 개발하였다.
처음으로 다른 사람이 구현해놓은 프로젝트를 본 것이었고,
코드 분석이란걸 해보았다.
다른 사람들이 어떤 생각으로 어떻게 코드를 구현했는지
알아볼 수 있었다.
오래된 코드를 보면서 리팩토링이란 것에도 많은 관심이 생겼었고,
나중엔 CRM4U 모바일버전(반응형)을 개발하게 되었을 때
프로젝트 몸체부터 공통 기능들을 맡게 되어,
어떻게 구현해놓으면 사람들이 쉽게 쓸 수 있을까를 많이 고민했었다.
자연스럽게 디자인패턴, 클린코드에 대해 공부를 하게 됐었다.
최대한 깔끔하게 만들려고 노력했었고, 애정이 많이 남은 프로젝트이다.

Automall4U

중고차 매입 / 판매 와 고객관리를 위한 CRM 구축

  • 기간: 2014.02 ~ 2014.11
  • 발주: KCC오토그룹

  • 주요기술:
  • Spring F/W(전자정부), Oracle DB

  • 주요역할:
  • 1. 사용자, 메뉴, 권한, 조직 등 시스템 주요 기본기능 구현
  • 2. CentOS 기반의 Web Server(Apache), WAS(Tomcat) 서버 구축

[ 첫 회사, 첫 프로젝트, 신입OJT ]
모든 것이 처음이었던 프로젝트
처음해보는 협업
서로 다른 역할일지라도 계속 생각을 나누지 않으면,
배가 산이 아닌 우주까지도 갈 수 있겠다라는 생각도 들었었다.
이를 해결할 수 있는건 커뮤니케이션과 리더쉽이란 것도 깨달았다.

또, Web Server는 무엇이고, WAS도 잘 모르지만 홀로 진행했던 서버 구축
구글링을 통해 계속 답을 찾아갔다. 아마 이 때, 구글링 실력이 제법 올라간 것 같다.
가장 어려웠던건 네트워크였다.
모든 설정들을 다 했을 때, 잘 되리라 생각했다.
하지만, 시스템은 정상적으로 돌지 않았고, 혼자 야근까지 하면서 며칠간 끙끙댔다.
결국, 회사 내부에는 네트워크팀이 있고
그 팀을 통해 네트워크끼리 접속 허용 연결을 요청해야 된다는 것을 알게되었다.
허무했지만, 이 때부터 네트워크에 대해 조금씩 이해하기 시작한거 같다.

이 프로젝트는 내게 많은 지식을 얻게 해준거 같다.
협업, 분석/설계, 구현, 서버, 네트워크 등 레벨이 높진 않지만
시스템의 기초를 알게 해준 프로젝트였다.
